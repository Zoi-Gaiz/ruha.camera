# RUHAcam 

![](ruhacam-cover.jpg)

## Bill of Materials 

![](ruhacam-bom.jpg)

- Raspberry Pi Zero W + microSD card 
- Raspberry Pi HQ Camera 
- 16mm 10MP Telephoto Lens 
- 2.2" TFT Display 
- 2000mAh Li-Pi battery 
- Power switch 
- Shutter button 
- Dupont jumper wires 
- 3D printed case (see STL folder)

## Schematic

| Raspberry pi pins | TFT display |
|-------------------|--------|
| 3v3 pin 17 | VCC |
| GND pin 20 | GND |
| BCM 8 pin 24 | CS |
| BCM 23 pin 16 | RESET |
| BCM 24 pin 18 | DC |
| BCM 10 pin 19 | MOSI |
| BCM 11 pin 23 | SCK | 
| BCM 18 pin 12 | LED | 
| BCM 9 pin 21 | MISO |
